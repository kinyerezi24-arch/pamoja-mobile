# PAMOJA Mobile - Flutter Foundation

**One Platform. Every Business.**

PAMOJA Mobile is the mobile-first application for the PAMOJA Business Operating System.

## Project Status

**Version:** 0.1.0 | **Status:** Foundation - ENG-001 | **Platform:** Flutter

## What's Included

✅ Flutter Stable  
✅ Material 3  
✅ Riverpod  
✅ GoRouter  
✅ Localization (EN/SW)  
✅ Light & Dark Themes  
✅ Google Fonts  
✅ Clean Architecture  
✅ Feature-First Structure  

## Quick Start

```bash
git clone https://github.com/kinyerezi24-arch/pamoja-mobile.git
cd pamoja-mobile
flutter pub get
flutter run
```
